### 1. clone this repository.
For Minimum Clone
```
$ git clone git@github.com:YumaNK/ashioto.git
```

For Entire Clone
```
$ git clone git@github.com:YumaNK/ashioto.git --recursive
```

### Frontend

```
yarn install
yarn run build
```

### Declaration
At this stage, do not trust any results generated by the web application. The selection of chemical compounds and the values of their chemical properties are randomly generated.
The possible SMILES strings are extracted from:
https://github.com/xxx/qcbm/blob/main/example/data/tiny.csv
